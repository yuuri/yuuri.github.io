一条nginx命令的思考

2019-09-04 1 min read 
2>&1 nginx  -V |tr ' ' '\n'

tr 是替换命令

nginx -V是显示nginx已安装模块命令

2>&1 是将错误输出重定向到标准输出 1就是屏幕

这里为什么需要加上 2>&1 呢？

如果不加这一句 nginx -V 是直接由nginx返回的 bash的命令是无法调用的

需要将nginx命令的结果封装一层 把结果给bash 才能执行bash的命令。
