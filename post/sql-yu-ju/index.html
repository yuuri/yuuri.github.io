2019-09-05 1 min read 
select sum(v_p.price) from student s where s.grade_id =7  join vip_list v_l on s.id =  v_1.student_id join vip_price v_p on v_l.name = v_p.name

where语句最后才进行判断，会把所有的条件都查出来最后才筛选，性能不好

使用子查询可以先筛选再进行多表联查

select sum(vip_price.price) from vip_price where vip_price.name IN
(select vip_list.name from vip_list where vip_list.student_id IN
(select student.id from student where student.grade_id='7'))

或者可以这样

select sum(v_p.price) from v_list v join student s on s.id = v.student_id and s.grade_id = 7 join v_price v_p on v.name = v_p.name
