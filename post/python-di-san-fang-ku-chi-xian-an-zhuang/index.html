Python第三方库离线安装

2019-05-14 1 min read

Python第三方库安装起来很简单，但是有的时候如果环境没有网络的话，处理起来还是很麻烦的。这里简单记录一下。

1.导出环境依赖

pip freeze > requirements.txt

2.根据依赖下载离线安装包

pip download -r requirements.txt -d packages

3.将 requirements.txt 和 packages 压缩一下，这里实例压缩成了 zip，将 zip 包一层层的发送到远程开发机中

4.安装离线包

pip install --no-index --find-links=pakages -r requirements.txt
